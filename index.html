Get Started

Run your first Troy program in less than 5 minutes.

Installation

The easiest way to install Troy under your user account is via virtualenv and easy_install or pip. 

$ virtualenv $HOME/troy
$ source $HOME/troy/bin/activate
$ easy_install Troy

More detailed installation instructions can be found in the manual.
